Build from source

  1. cd kuni && npm install && npm run build
  2. Desktop: cd desktop && npm install && npm run build → .dmg / .exe
  3. Android: npx cap add android && npm run android:sync then open Android Studio → APK

Requires WebGPU on the device for KunoEngine inference. First model load downloads ~6GB weights.
