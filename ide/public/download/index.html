Android · install as app (PWA)

Juni does not ship a Play Store APK. Install the web IDE from Chrome with Add to Home Screen / Install app for an app-like icon and standalone window.

  1. Open the Juni web IDE in Chrome on your phone.
  2. Tap the menu (⋮) → Install app or Add to Home screen.
  3. Confirm. Launch Juni from your home screen like any other app.

Requires HTTPS, a web manifest, and a service worker (already included). On iOS Safari: Share → Add to Home Screen.
