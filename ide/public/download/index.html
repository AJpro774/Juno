Android

APK — download the latest Juni APK from GitHub Releases and sideload (enable Install unknown apps for your browser/file manager). Prefer the release asset named Juni-IDE-android.apk (built by the Release Android APK GitHub Action / Bubblewrap TWA).

PWA (optional) — install the web IDE from Chrome with Add to Home Screen / Install app for an app-like icon without sideloading.

  1. Open the Juni web IDE in Chrome on your phone (PWA path).
  2. Tap the menu (⋮) → Install app or Add to Home screen.
  3. Or download the APK from Releases and open the file to install.

PWA requires HTTPS, a web manifest, and a service worker (already included). On iOS Safari: Share → Add to Home Screen (no APK on iOS).
