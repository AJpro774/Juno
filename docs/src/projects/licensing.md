# Licensing

Juni / Juno uses a clear split:

| What | Terms |
|------|--------|
| **Source code** in this repository | [Apache License 2.0](../../LICENSE) only |
| **Distributed IDE / apps** (web IDE, desktop installers, PWA) | [EULA](../../EULA.md) **and** Apache-2.0 for the underlying source |

In the IDE:

1. **First launch** — accept the EULA (stored locally as `juni.eula.accepted.v1`).
2. **Settings → Legal** — open the full EULA or Apache License.
3. **Credits** — includes a short license summary.

Your Juni projects and games remain yours (see EULA).
